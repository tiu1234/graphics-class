This program can only run on linux system.
	In case you do not have SDL packet.
		If your machine is Xubuntu
			sudo apt-get install libsdl1.2-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libsdl-ttf2.0-dev
		If your machine is red hat
			sudo yum install SDL-devel SDL_mixer-devel SDL_image-devel SDL_ttf-devel
'make' to make
'make clean' to clean
'q' to switch between texture mode and cube mode.
drag the mouse to rotate the teapot.
