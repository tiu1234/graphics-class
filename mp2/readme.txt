'make clean' to clean the build
'make' to build
'./mp2' to run the flying simulation
'a' to accelarate (speed up to 0.03)
'd' to decelarate (speed down to 0.005)
up arrow key to pitch down
down arrow key to pitch up
left arrow key to roll left
right arrow key to roll right
'q' to turn left horizontally
'e' to turn right horizontally
demo video link: http://youtu.be/gkxc4EBY-a0
